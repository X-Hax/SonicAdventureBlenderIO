The addon provides a lot of tools, all of which can be accessed through the panels in this 3D View toolbar.

the toolbar encompasses following panels:

  • Import: Importing data from files.
  • Export: Exporting data to files.
  • Material: Various material utilities.
  • Utilities: All sorts of utilities.
  • Material Mass Edit: For quickly editing material data.
  • Migration: Tools to migrate blender projects from the old addon.
  • Info: Open various internet resources.
